# Smart Online Ordering and Management System (Backend)

## 🚀 Overview

A scalable backend REST API built using Spring Boot to manage an online food ordering system. It handles user authentication, product operations, and order processing with secure and structured architecture.

## 🛠️ Tech Stack

* Java
* Spring Boot
* Spring Security (JWT)
* JPA / Hibernate
* MySQL

## 🔐 Features

* User Authentication & Authorization (JWT-based)
* Role-based access control (Admin/User)
* Secure REST API endpoints
* Product management
* Order processing system

## 📁 Project Structure

* controller → API endpoints
* service → business logic
* repository → database layer
* entity → data models
* config → security configuration

## ⚙️ Setup & Run

1. Clone the repository
2. Configure database in `application.properties`
3. Run Spring Boot application
4. Test APIs using Postman

## 🧪 API Testing

All endpoints are tested using Postman.
